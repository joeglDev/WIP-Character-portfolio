# Character portfolio

A personal project to display user generated fictional characters in an aesthetic manner -> an art portfolio.

## Stack

- Language - TypeScript
- Back-End - Node.js / express.js (server), passport.js (authentification), mongoDB (noSQL database)
- Front-End - React
